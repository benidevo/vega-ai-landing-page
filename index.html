Installation Instructions

  1. 1. Download the extension from GitHub
  2. 2. Open Chrome/Edge and go to extensions (chrome://extensions)
  3. 3. Enable "Developer mode" in the top right
  4. 4. Click "Load unpacked" and select the extension folder

Set up Vega in minutes with Docker and begin navigating toward your dream career

Quick Setup Guide

1

Clone the Repository

git clone https://github.com/benidevo/vega.git
cd vega
2

Configure Environment

cp .env.example .env # Edit .env with your API keys
3

Start with Docker

docker-compose up -d # Or use: make run
4

Access Vega

Open your browser and navigate to:

http://localhost:8000

Requirements & Resources
